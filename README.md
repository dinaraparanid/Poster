## **About App**
Multiplatform mini-social-network application
that allows to share posts, written in Flutter.

Application utilizes principle of **Clean Architecture**,
using **BLoC** library as a state management tool.

## **Structure**
Application utilizes principle of **Clean Architecture**,
using **Bloc** library as a state management tool.
